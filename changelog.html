Changelog

Reverse-chronological log of meaningful updates to the OTM-Agent project and public site.

2026-05-21
v1.0 — Full production readiness achieved
All 4 v1.0 benchmarks live with real results. GitHub Actions workflows for automated sync. Terraform module for Cloud Run benchmark infrastructure. Paper methodology draft complete.
2026-05-21
Week 5: OTM-OOS-Replay benchmark — headline result published
Day-by-day walk-forward over 261 trading days (2025-05-01 to 2026-05-01). Directional accuracy: 53.4% [CI: 50.7%–56.2%] vs 50.0% random baseline. Sharpe, cumulative P&L, max drawdown, and per-month breakdown published. Causal filter (query_ts) enforced throughout.
2026-05-21
Week 4: Skill-Retention + Mem-Utility benchmarks live
OTM-Skill-Retention: 88.0% pass rate [CI: 84.5%–91.5%] across 20 skill probes at delays of 1–10 turns. Per-category and per-delay breakdowns published. OTM-Mem-Utility: +0.716 memory utility [CI: 0.673–0.759] — memory retrieval improves outcomes across all query categories. Historical comparison queries benefit most.
2026-05-21
Week 3: Automation infrastructure deployed
GitHub Actions: sync-results.yml (6h cron), lint.yml (PR validation). GCS sync script for pulling results. Cloud Run Job + 4 Cloud Scheduler triggers (weekly, bi-weekly, monthly). Terraform module with Pub/Sub notifications. Upload script for archiving results.
2026-05-21
Week 2: First benchmark — OTM-Persona-Align
Benchmark framework created (base.py ABC + 4 runners + CLI). OTM-Persona-Align: cosine divergence 0.402 [CI: 0.351–0.457] between execution and analysis personas. Baseline (neutral mode): 1.006. Skill Jaccard distance confirms differentiated behavior.
2026-05-21
v0.1 — Public site launched on GitHub Pages
Site scaffolded with 8 pages: home, architecture, methods, benchmarks, live dashboard, audit cycle, changelog, and about. Static HTML/CSS, dark mode default, deployed via GitHub Pages. All content drawn from the OTM-Agent Technical Report and production readiness plan.
2026-05-21
Production readiness plan finalized
6-week plan covering public site, 4 benchmarks (persona-align, skill-retention, mem-utility, OOS replay), continuous evaluation via Cloud Run Jobs + GitHub Actions, and honest reporting of all results including negative findings.
2026-05-20
Strategy v2 designed — walk-forward ML composite
Root cause analysis of Sharpe −0.10 identified 5 problems in the naive backtest. Strategy v2 implements regime-weighted ML composite (65% regime classifier at 60% weight), confidence gating (0.58 threshold), and 5% drawdown stop. Pending BQ execution.
2026-05-19
Causal leakage prevention — 16 tests passing
Threaded query_ts through all 8 memory retrieval paths. 16 dedicated tests verify local cache filtering, backward compatibility, and BQ query source analysis. Prevents agent from "remembering the future" during OOS replay.
2026-05-18
OOS holdout audit completed
Static analysis scanned 48 training-table queries. Found 20+ missing the date filter — all corrected. Training tables had 0 rows at time of audit, so no model was ever trained on leaked data. All models trained from scratch with filters in place.
2026-05-17
256 tests passing — full test suite green
Test suite covers OOS leakage (5 tests), causal leakage (16 tests), skill registry validation, memory tier operations, persona Bayesian updates, reward computation, plan validation, and integration flows.
2026-05-15
Live SAR dashboard deployed
Single-page HTML dashboard pulling live data from Cloud Run API. Shows per-tank radar signals, region summaries, daily trends, oil price correlation, and anomaly detection across all 4 global hubs.
2026-05-12
Stage 3 pipeline operational
Signal generation, paper trading, and governance scorecard deployed on Cloud Run. Daily composite trading signals with confidence levels. Signal API and GOII API live.
2026-05-08
Stage 2 ML models — walk-forward evaluation
Regime classifier: 65.35% (vs 52.9% baseline). Spread forecaster: 51.4%. EIA surprise: 51.52%. All trained with OOS filter, random_state=42, model version v0.2.0-oos-clean.
2026-05-01
Stage 1 Kalman pipeline — 309 hub-state days
5,124 SAR observations processed for Cushing. Kalman filter converges from variance 0.066 to 0.045. EIA direction accuracy: 52.8%. All 5 Stage 1 exit criteria passed.
